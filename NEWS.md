# asciiwright news

## 0.16.0

- Added snapshot-style regression tests for representative ASCII Wright map outputs, including paged and polytomous layouts.
- Added `NEWS.md` and refreshed the development plan to make release history and next steps easier to track.

## 0.15.0

- Extended `line_length`, `max_page`, and `table_style` support to polytomous threshold and range maps.
- Added Winsteps-style polytomous presets for Table 1.4 through 1.8.

## 0.14.0

- Added Winsteps table presets, side swapping, mirrored easiness layouts, `line_length`, and pagination to the core Wright map renderer.
- Refined score extreme handling and density-column note wording.

## 0.13.0

- Added a vignette for the label preview and override workflow.
- Added GitHub Actions for `R CMD check` and a README status badge.

## 0.12.0

- Added `preview_label_abbrev()` and `make_label_overrides()`.
- Added a development plan and pushed the project to GitHub.

## 0.11.0

- Added `wright_map_mirt()` for single-dimension `mirt` item, threshold, and range maps.

## 0.10.0

- Added side-aware custom label override support for standard and range maps.

## 0.9.0

- Added Table 1.4-style polytomous range maps.

## 0.8.0

- Added Table 1.5 to 1.8-style polytomous threshold maps.

## 0.7.0

- Added `score_table_ascii()` for Winsteps-style raw score to measure tables.

## 0.6.0

- Added Rasch-based raw-score placement for the `SCORE` column.

## 0.5.0

- Added export helpers for `.txt` and `.md` output.

## 0.4.0

- Added mirrored double-axis layouts and optional right-side `MEASURE` columns.

## 0.3.0

- Added Winsteps-like density symbol rules and grouped distribution notes.

## 0.2.0

- Added distribution-column rendering modes and improved axis-aligned labels.

## 0.1.0

- Initial prototype for terminal-friendly ASCII Wright maps in R.
